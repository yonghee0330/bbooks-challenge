Firebase 연결이 필요해요

이 파일 상단의 FIREBASE_CONFIG 값을 Firebase 콘솔에서 복사해 넣어주세요.

// 이 파일 상단 <script> 안에서
// FIREBASE_CONFIG 객체를 채워주세요:

const FIREBASE_CONFIG = {
  apiKey: "여기에 붙여넣기",
  authDomain: "...",
  projectId: "...",
  ...
};

Firebase 콘솔 → 프로젝트 설정 → 앱 추가(웹) → SDK 설정에서 firebaseConfig 값을 복사하세요.

Authentication → 로그인 방법 → 익명(Anonymous)을 사용 설정하세요.

Firestore Database도 생성하고, 규칙을 아래로 설정해주세요:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

익명 로그인된 사용자만 읽기/쓰기 가능 — 완전 공개보다 안전합니다.
